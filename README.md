 # Jenny's Desk Setup Website

Currently developing my personal website using Vite, vanilla JS, Three.js, ...

### Local setup
1. `git clone https://github.com/jennyho1/jennyho1.github.io`
2. `cd jennyho1.github.io`
3. `npm install`
4. Run app in development mode on http://localhost:3000 `npm run dev`

### You can view the website [here](https://jennyho1.github.io/)
